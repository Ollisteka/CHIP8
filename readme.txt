CHIP8 emulator
Поиграй в игры 30-летней давности!


Справка по запуску:
    Windows: python client.py --help
    Linux: python3 client.py --help
Пример запуска: python client.py games/MAZE


MAZE:
рисует на экране случайный лабиринт

KALEID:
программа для рисования узоров.

Управление:
-стрелочки - передвижение и рисование по экрану (вверх, вниз, вправо, влево)
-CTRL - CHIP8 повторит ваши движения за вас!
